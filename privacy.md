# Privacy Policy — Talks by Dave Publisher

_Last updated: April 15, 2026_

## 1. About This Tool

Talks by Dave Publisher ("the Tool") is a personal, single-user desktop
application operated by David Smith. It is not offered as a service to any
other person or organization, and it does not collect data from anyone other
than its operator.

## 2. What Information the Tool Handles

The Tool handles only the following information, all belonging to its single
operator:

- **Video files** that the operator places in their own Google Drive folder.
- **Generated captions, titles, and hashtags** created from those videos.
- **TikTok API credentials** (Client Key, Client Secret, and access tokens)
  that the operator provides so the Tool can publish to the operator's own
  TikTok account.
- **Google Drive API credentials** that the operator provides so the Tool
  can read from the operator's own Drive folder.

## 3. Where Information Is Stored

All information stays on the operator's personal computer:

- Videos, captions, and metadata are stored in local folders on the
  operator's machine.
- API credentials are stored in a local configuration file on the operator's
  machine.

No data is sent to any server, cloud service, analytics provider, or third
party operated by the Tool's author, except:

- **TikTok**, when publishing the operator's own approved videos.
- **Google Drive**, when reading the operator's own source videos.

Both interactions are direct API calls initiated by the Tool on the
operator's computer.

## 4. What We Do Not Do

- The Tool does not collect information from any user other than its
  operator.
- The Tool does not share, sell, or transmit data to any third party beyond
  the TikTok and Google Drive APIs needed to perform its core function.
- The Tool does not use cookies, trackers, or analytics.
- The Tool does not have accounts, sign-ups, or user profiles.

## 5. Data Retention

The operator controls all data on their own machine and may delete any file,
credential, or record at any time. There is no centralized storage to
retain.

## 6. Security

API credentials are stored in a local environment file and are not logged,
transmitted, or printed by the Tool. The operator is responsible for the
physical security of their own computer.

## 7. Children

The Tool is not directed to, or used by, anyone under 13.

## 8. Changes

This policy may be updated at any time. The "Last updated" date at the top
reflects the most recent change.
